### 操作系统第三次作业

#### 第一题

##### （1）

地址空间有$2^{32}bit=4GB$。

一页的大小是$2^{12}bit=4KB$。

##### （2）

第一级页表的起始逻辑地址是`0x80000000`。

由于`0x80000000`的二进制为`1000 0000 0000 0000 0000 0000 0000 0000`，前10位位页目录号，即`1000 0000 00 `(二进制) = `512 `(十进制)，故第一级页表的也框号为`512`。

##### （3）

* 0x0：页目录索引（0000000000）为0，对应页目录项为0x0，物理页框号为0x0，有效位为0，即无效。

* 0x00803004：

  页目录索引（0000000010）为2，对应页目录项为0x5001，物理页框号为0x5，有效位为1。

  页表号（0000000011）为3，在基值为0x5 * 0x1000 = 0x5000的页表中查找得到对应页表项为0x20001，物理页框号为0x20，有效位为1。

  页内偏移（000000000100）为4，故所得对应物理地址为0x20 * 0x1000 + 0x4 = 0x00020004

* 0x00402001：

  页目录索引（0000000001）为1，对应页目录项为0x1001，物理页框号为0x1，有效位为1。

  页表号（0000000010）为2，在基值为0x1 * 0x1000 = 0x1000的页表中查找得到对应页表项为0x5001，物理页框号为0x5，有效位为1。

  页内偏移（000000000001）为1，故所得对应物理地址为0x5 * 0x1000 + 0x1 = 0x00005001

##### （4）

* 物理地址为0x326028，物理页框号为0x326，页内偏移为0x028

* 在图中，物理地址为0x20000的页表的第1项记录了0x326001，即物理页框为0x326，有效位为1，符合所需的物理地址的页框。该页表所在的地址对应的物理页框为0x20000 / 0x1000 = 0x20。

* 在图中，一级页表的第3项记录了0x20001，即物理页框为0x20，有效位为1，符合所需的物理地址的页框。该页表所在的地址对应的物理页框为0x5000 / 0x1000 = 0x5。

* 故逻辑地址为：

  * 一级页表号：3 = 0000_0000_11
  * 二级页表号：1 = 0000_0000_01
  * 页内偏移：0x328

  综上：逻辑地址为0x00c01328

#### 第二题

| 指令               | 结果  |
| ------------------ | ----- |
| Load [0x00001022]  | 0x20  |
| Store [0x00C07222] | Error |
| Store[0x00C005BF]  | OK    |
| Load [0x00003013]  | 0x19  |
| Load [0xFF80078F]  | Error |
| Load [0xFFFFF005]  | 0x66  |

* 0x00001022：一级页表号：0，二级页表号：1，页内偏移：0x022

  在0x0020_0000中访问一级页表[0]，得到0x0010_0007。即物理页框号0x00100，读写位1，有效位1。

  在0x00100 * 0x1000 = 0x0010_0000中访问二级页表[1]，得到0x0000_4067。即物理页框号0x00004，读写位1，有效位1

  计算物理地址：0x00004 * 0x1000 + 0x022 = 0x0000_4022，访问得到0x20。

* 0x00C07222：一级页表号：3，二级页表号：7，页内偏移：0x222

  在0x0020_0000中访问一级页表[3]，得到0x0010_3007。即物理页框号0x00103，读写位1，有效位1。

  在0x00103 * 0x1000 = 0x0010_3000中访问二级页表[7]，得到0xEEFF_0001。即物理页框号0xEEFF0，读写位0，有效位1

  只读不写故Store失效。

* 0x00C005BF：一级页表号：3，二级页表号：0，页内偏移：0x5BF

  在0x0020_0000中访问一级页表[3]，得到0x0010_3007。即物理页框号0x00103，读写位1，有效位1。

  在0x00103 * 0x1000 = 0x0010_3000中访问二级页表[0]，得到0x1122_0067。即物理页框号0x11220，读写位1，有效位1

  计算物理地址：0x11220 * 0x1000 + 0x5BF = 0x1122_05BF，写入成功。

* 0x00003013：一级页表号：0，二级页表号：3，页内偏移：0x013

  在0x0020_0000中访问一级页表[0]，得到0x0010_0007。即物理页框号0x00100，读写位1，有效位1。

  在0x00100 * 0x1000 = 0x0010_0000中访问二级页表[3]，得到0x0000_4007。即物理页框号0x00004，读写位1，有效位1

  计算物理地址：0x00004 * 0x1000 + 0x013 = 0x0000_4013，访问得到0x19。

* 0xFF80078F：一级页表号：（二进制）1111_1111_10，二级页表号：0，页内偏移：0x78F

  在0x0020_0000 + 0xFF = 0x0020_0FF0访问一级页表[2]，得到0x001F_E007。即物理页框号0x001FE，读写位1，有效位1。

  在0x001FE * 0x1000 = 0x001F_E000中访问二级页表[0]，得到0x0415_0000。即物理页框号0x04150，读写位0，有效位0

  无效故Load失败。

* 0xFFFFF005：一级页表号：（二进制）1111_1111_11，二级页表号：1111_1111_11，页内偏移：0x005

  在0x0020_0000 + 0xFF = 0x0020_0FF0中访问一级页表[3]，得到0x001F_F007。即物理页框号0x001FF，读写位1，有效位1。

  在0x001FF * 0x1000 + 0xFF0 = 0x001F_FFF0中访问二级页表[3]，得到0x0010_3067。即物理页框号0x00103，读写位1，有效位1

  计算物理地址：0x00103 * 0x1000 + 0x005 = 0x0010_3005，访问得到0x66。
